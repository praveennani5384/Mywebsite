Naukri.com Job Scraper

Detailed Explanation of the System

This project is a web scraper that extracts job details from Naukri.com, a popular Indian job portal. It utilizes Python with the Selenium library to automate web browser interactions and extract job titles, company names, and locations. The extracted data is then saved to a CSV file for further analysis.

Core Functionalities

  • Automated Web Scraping: Uses Selenium to control a Chrome browser, navigates to Naukri.com, and automatically extracts job data.
  • Multiple Page Scraping: Scrapes data from multiple pages of job listings by automatically clicking the "next" button. It iterates through the first 10 pages.
  • Dynamic Data Extraction: Uses XPath to locate the required HTML elements (job titles, company names, and locations) and extracts their text content.
  • Robust Error Handling: Includes a try-except block to gracefully handle any unexpected errors during scraping, preventing script crashes.
  • Data Storage: Saves the extracted job details to a CSV file (`job_details.csv`) with proper headers for easy data analysis.
  • User-Friendly Output: Provides console output showing the progress of the scraping and confirms when the CSV export is done.
  • Explicit Waits: Includes `time.sleep()` to handle website loading times, reducing the chance of errors due to elements not being immediately available.
  • Headless Browser: The script uses Chrome Options to ignore certificate and SSL errors, increasing the robustness of the scraper.
  • Action Chains: Uses Selenium's ActionChains to interact with the 'next' page button to handle cases where a simple click does not work.
