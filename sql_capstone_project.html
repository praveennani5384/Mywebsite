SQL Capstone Project

This project involves performing various analytical queries on NFT sales data using SQL. The data includes information about CryptoPunk sales, such as sale prices, dates, buyer addresses, and seller addresses. The goal was to extract meaningful insights from this data using complex SQL queries.

Project Description

The dataset utilized in this project pertains to CryptoPunk NFT sales, encompassing details such as transaction dates, prices in both ETH and USD, as well as the addresses of both buyers and sellers. The primary objective was to leverage SQL for in-depth analysis, thereby extracting significant patterns and business-oriented insights.

Key Queries and Insights

  • Total Sales Count: Determined the total number of sales within a specified date range.
  • Top 5 Transactions: Identified the top 5 most expensive transactions based on USD price, and also extracted the name, ETH price, USD price and date of the transactions
  • Moving Average Price: Calculated the moving average of USD price over the last 50 transactions which provided trend analysis on the price changes.
  • Average Sale Price by NFT: Determined each NFT's average sales price, sorted in descending order, highlighting the top-performing NFTs.
  • Sales by Day of the Week: Provided analysis of sales by each day of the week and their average price in ETH, ordered by the number of transactions.
  • Transaction Summary: Constructed a descriptive summary of each transaction with all the relevant details such as name of the NFT, buyer, seller, date and the price.
  • View Creation: Created a database view “1919_purchases” to filter the sales from a given address.
  • ETH Price Histogram: Categorized ETH price ranges, creating a histogram to illustrate price frequency.
  • High and Low Sale Prices: Generated a unified query that shows the highest and lowest purchase prices for each NFT.
  • Monthly Top NFT Sales: Determined the NFT with the highest sales count and the maximum sale price for a given month and year.
  • Monthly Total Volume: Calculated the total sales volume on a monthly basis.
  • Wallet Transaction Count: Determined number of sales made by a given wallet address as a buyer, seller and total transactions.
  • Estimated Average Value Calculation: Built a robust value calculator that filters out outliers in daily sales and calculates a representative average price.
  • Wallet Profitability: Determined the overall profitability (or loss) of each wallet, ordered by the most profitable.
