Credit Card Fraud Detection

This project focuses on building a machine learning model to detect fraudulent credit card transactions. The project includes data preprocessing, exploratory data analysis, model training, hyperparameter tuning, and evaluation steps. A random forest classifier is used in this project to make predictions.

Project Description

The aim of this project is to develop a robust model capable of identifying fraudulent credit card transactions using a dataset consisting of various transaction features including Time, V1 to V28, and Amount. The dataset is highly imbalanced (fraudulent transactions being very low compared to legitimate ones). The model should be able to accurately detect fraud to prevent financial losses.

Key Steps and Features

  • Data Loading and Exploration: Loading the dataset, exploring its structure, understanding the features, and checking for any missing values and descriptive statistics.
  • Data Preprocessing: Converting the time variable into a day format and performing data transformation of the features for better model performance.
  • Data Balancing: Implementing undersampling to balance the skewed data to reduce the bias of the data.
  • Data Split: Splitting the dataset into training and testing sets.
  • Model Training and Evaluation: Train, evaluate, and compare various models like logistic regression, decision tree, random forest, svm and k nearest neighbours.
  • Model Selection and Optimization: Select the Random forest model as the main model and use GridSearchCV to tune the parameters to get the best performance.
  • Model Validation: Test the performance of the model using metrics like precision, recall, f1-score and ROC curve and confusion matrix.
  • Random data generation and prediction: Created a script to automate generating random data and making the prediction, that data is stored in an excel sheet.
  • Pipeline Creation: Created a pipeline for scaling the data and prediction.
