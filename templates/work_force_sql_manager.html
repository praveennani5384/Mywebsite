SQL Workforce Management

Detailed Explanation of the System

This project implements a web-based Employee Management System using Flask, SQLAlchemy, and SQLite. The system allows users to perform basic CRUD (Create, Read, Update, Delete) operations on employee records, filter employees based on various criteria, and export filtered data to CSV. Additionally, it includes a Selenium-based automation script to populate the system from an Excel file.

Core Functionalities

  • Add Employee: Allows users to input employee ID, name, position, and salary through a form. Data is validated (salary must be a float) before being saved in the database. Handles duplicate IDs by providing a flash message.
  • Remove Employee: Allows users to remove employees by entering their ID. Handles cases where an employee ID does not exist, by showing flash message.
  • Promote Employee: Allows users to increase an employee's salary by entering their ID and the amount to increase. Handles invalid amounts and non-existent employee IDs.
  • Display Employees: Displays all employee records in a tabular format on a single page.
  • Find Employee: Allows users to search for a specific employee by their ID. Displays employee details if found; otherwise, displays an error message. Provides an edit button to edit the details of the employee.
  • Edit Employee: Allows editing of an employee's name, position, and salary. Handles invalid salary inputs. Redirects to the employee display page after a successful update.
  • Filter Employees: Allows filtering based on:
    • Position
    • Salary greater than a specified value
    • Salary less than a specified value
    • Order by salary (ascending or descending)
    Displays filtered data and provides an option to export it to a CSV file.
  • Export Filtered Employees: Exports filtered employee data to a CSV file.
  • Selenium Automation: Uses Selenium to automate the population of data in the system using data from an excel file (`sample_data.xlsx`). Automates both addition and removal operations from the excel sheet.
