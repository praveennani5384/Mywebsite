Movie Recommendation System

This project implements a content-based movie recommendation system using Python, Pandas, scikit-learn, and Streamlit. The system is designed to recommend movies based on the similarity of their content, such as overview, genres, keywords, cast, and crew. It uses a dataset of movie information to train the model and generate recommendations.

Project Description

The movie recommendation system leverages a dataset containing movie details like descriptions, genres, keywords, cast, and crew members. The core idea behind the project is to preprocess and transform the text-based movie attributes into numerical data that a machine learning algorithm can understand. This allows for determining semantic similarity between different movies to make personalized recommendations. The model uses cosine similarity to measure movie similarity, and the Streamlit application then provides an interface to interact with the model.

Key steps and Features

  • Data Loading and Merging: Loads movie and credits data, merging them to have a combined dataset.
  • Feature Selection: Selects relevant columns such as 'movie\_id', 'title', 'overview', 'genres', 'keywords', 'cast', and 'crew'.
  • Data Cleaning and Preprocessing: Handles missing values, converts textual data into lists, extracts director from crew information and removes spaces.
  • Tag Generation: Combines overview, genres, keywords, cast, and crew to create the 'tags' column for similarity calculation.
  • Text Transformation: Converts the tags into lower case and applies stemming to reduce words to their root forms.
  • Text Vectorization: Uses CountVectorizer to convert the text data into a matrix of token counts, and removes stop words.
  • Cosine Similarity: Computes cosine similarity matrix based on vectorized data, identifying the similarity between the movie attributes.
  • Recommendation Function: Implements a function that takes the movie title as input, identifies similar movies and provide a list of recommendations.
  • Streamlit Application: Provides a user-friendly interface via Streamlit, allowing users to select a movie and view the recommendations.
