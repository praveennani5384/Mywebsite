E-commerce Sales Analysis

This project implements an e-commerce sales analysis system that provides sales forecasting, customer segmentation, and product recommendations. It is developed using Python, Pandas, statsmodels, scikit-learn, and Streamlit.

Core Functionalities

  • Sales Forecasting: Utilizes an ARIMA model to predict future sales based on historical data. Includes plotting the forecasted results along with the test data.
  • Customer Segmentation: Segments customers into distinct clusters using K-Means clustering based on RFM (Recency, Frequency, Monetary) values. Allows visualization of cluster characteristics.
  • Product Recommendation: Implements a content-based recommendation system to suggest products to individual customers based on their purchase history.
  • Streamlit Application: A web application was developed to provide a user-friendly interface for the sales analysis with features such as a dynamic selection box, different plot types etc.
