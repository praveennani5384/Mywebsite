MNIST Digit Recognition

Detailed Explanation of the Project

This project implements a Convolutional Neural Network (CNN) to recognize handwritten digits from the MNIST dataset. The project consists of two main parts: a training script (`mnist.ipynb`) and a testing script (`testing_mnist.ipynb`). The training script loads the MNIST dataset, builds and trains the CNN, and saves the trained model. The testing script loads this saved model and uses it to predict the digit in a custom image.

Training the CNN (mnist.ipynb)

  • Data Loading and Preprocessing:
    • Loads the MNIST dataset using tensorflow.keras.datasets.mnist, it provides 60,000 training images and 10,000 test images.
    • Normalizes pixel values to a range between 0 and 1 by dividing them by 255.0.
    • Reshapes the data from 2D images to 4D tensors suitable for CNN input with the shape: (num_samples, 28, 28, 1).
  • CNN Model Definition:
    • Constructs a sequential CNN using TensorFlow and Keras.
    • The CNN consists of two convolutional layers (`Conv2D`) followed by max-pooling layers (`MaxPooling2D`). Each convolutional layer uses ReLU activation.
    • After the convolutional layers, there's a flattening layer, a dense layer with ReLU activation and dropout to avoid overfitting, and finally a dense output layer with 10 units and softmax activation for digit classification.
  • Model Compilation and Training:
    • The CNN is compiled using the Adam optimizer, sparse categorical crossentropy loss, and accuracy metrics.
    • The model is trained for 10 epochs using training data and validated with test data. It also uses the batch size of 64.
  • Visualization:
    • The training history is plotted to visualize the training and validation accuracy and loss curves with matplotlib.
  • Model Saving:
    • The trained model is saved to a file named digit_recognition_model.h5.

Testing the Model (testing_mnist.ipynb)

  • Model Loading:
    • Loads the pre-trained model from the file digit_recognition_model.h5.
  • Custom Image Prediction:
    • Defines a function called predict_custom_image that:
      • Loads an image from the specified path, converts it to grayscale and resizes it to 28x28 pixels.
      • Converts the image to a NumPy array, normalizes the pixel values.
      • Uses the loaded CNN model to predict the digit in the image and displays the digit and the confidence scores.
